1. **residential.png** - Image of a residential house
2. **commercial.png** - Image of a commercial building
3. **mixed-use.png** - Image of a mixed-use building

- Size: 200x200px or similar square dimensions
- Style: Illustrations matching the app's design theme (blue/purple accent colors)
